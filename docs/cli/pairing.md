# `openx pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:
- Pairing flow: [Pairing](/start/pairing)

## Commands

```bash
openx pairing list whatsapp
openx pairing approve whatsapp <code> --notify
```
