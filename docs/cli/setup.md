# `openx setup`

Initialize `~/.clawdbot/openx.json` and the agent workspace.

Related:
- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
openx setup
openx setup --workspace ~/clawd
```

To run the wizard via setup:

```bash
openx setup --wizard
```
