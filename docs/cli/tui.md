# `openx tui`

Open the terminal UI connected to the Gateway.

Related:
- TUI guide: [TUI](/tui)

## Examples

```bash
openx tui
openx tui --url ws://127.0.0.1:18789 --token <token>
openx tui --session main --deliver
```
